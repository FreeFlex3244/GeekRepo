Docker + Portainer + Watchtower

Must-Have

L'automatisation ultime. Watchtower met à jour vos containers pendant que vous dormez.

services:
  watchtower:
    image: containrrr/watchtower
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
    command: --cleanup --interval 3600
02

Le Ghost Network

OPNsense

Stable

Adieu les Box FAI limitées. Un routeur pro sur mini-PC pour un contrôle total du trafic.

AdGuard Home

Security

DNS Récursif local. Pas de Google DNS, pas de Cloudflare. Publicités bloquées au niveau du réseau.

services:
  adguardhome:
    image: adguard/adguardhome
    container_name: adguardhome
    ports:
      - 53:53/tcp
      - 53:53/udp
      - 3000:3000/tcp
    volumes:
      - /path/to/workdir:/opt/adguardhome/work
      - /path/to/confdir:/opt/adguardhome/conf
    restart: unless-stopped

CrowdSec

IPS

L'immunité collective. Bannit automatiquement les IPs agressives qui scannent vos ports.

services:
  crowdsec:
    image: crowdsecurity/crowdsec:latest
    container_name: crowdsec
    environment:
      - COLLECTIONS=crowdsecurity/linux crowdsecurity/sshd
    volumes:
      - /var/log:/var/log:ro
      - /etc/crowdsec:/etc/crowdsec
    restart: unless-stopped

Headscale

VPN

Réseau Mesh privé. Accédez à vos services de n'importe où sans ouvrir un seul port sur votre routeur.

services:
  headscale:
    image: headscale/headscale:latest
    container_name: headscale
    volumes:
      - /path/to/config:/etc/headscale
    ports:
      - 8080:8080
    command: headscale serve
    restart: unless-stopped

L'Empire du Multimédia

La Pile 'ARR' (Prowlarr, Radarr, Sonarr, Bazarr)

Automation

L'usine de traitement multimédia. Indexation, recherche, téléchargement et renommage automatique.

services:
  sonarr:
    image: lscr.io/linuxserver/sonarr:latest
    container_name: sonarr
    environment:
      - PUID=1000
      - PGID=1000
    volumes:
      - /path/to/data:/config
      - /path/to/tvseries:/tv
      - /path/to/downloadclient-downloads:/downloads
    ports:
      - 8989:8989
    restart: unless-stopped

Jellyfin

Media Server

L'alternative 100% libre à Plex. Pas de compte tiers, pas de tracking, juste votre média.

services:
  jellyfin:
    image: jellyfin/jellyfin
    container_name: jellyfin
    user: 1000:1000
    network_mode: 'host'
    volumes:
      - /path/to/config:/config
      - /path/to/cache:/cache
      - /path/to/media:/media
    restart: 'unless-stopped'

Real-Debrid + Zurg/Rclone

God-Tier Tip

La Citadelle Domotique (Offline First)

Home Assistant

Stable

Le cerveau de la maison. Zéro Cloud, contrôle total sur vos données et vos devices.

services:
  homeassistant:
    container_name: homeassistant
    image: "ghcr.io/home-assistant/home-assistant:stable"
    volumes:
      - /path/to/config:/config
      - /etc/localtime:/etc/localtime:ro
    restart: unless-stopped
    network_mode: host

Zigbee2MQTT

Open Standard

Dégagez les ponts propriétaires (Hue, IKEA). Un seul dongle USB pour tout contrôler en local.

services:
  zigbee2mqtt:
    container_name: zigbee2mqtt
    image: koenkk/zigbee2mqtt
    volumes:
      - /path/to/data:/app/data
    devices:
      - /dev/ttyACM0:/dev/ttyACM0
    restart: unless-stopped

Frigate NVR

AI Enhanced

Détection d'objets en temps réel via Google Coral. Vos caméras deviennent intelligentes sans envoyer une seule image dans le cloud.

services:
  frigate:
    container_name: frigate
    image: ghcr.io/blakeblackshear/frigate:stable
    shm_size: "64mb"
    devices:
      - /dev/bus/usb:/dev/bus/usb # For Coral
    volumes:
      - /path/to/config:/config
      - /path/to/media:/media/frigate
    ports:
      - "5000:5000"
    restart: unless-stopped
